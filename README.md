# Migrant Deaths Analysis

This repo contains all the code and files to generate the [findings](https://tucson.com/news/local/migrant-deaths-data-analysis-and-limitations/article_a385a946-f964-11eb-ac2e-23f8af9cea0e.html) for the Star's 2021 project on migrant deaths.

## Environments

I suggest using [anaconda](https://docs.anaconda.com/anaconda/user-guide/tasks/install-packages/) environments to manage multiple virtual environments as `empty_border_by_year_by_corridor.ipynb` will need its own.

Otherwise, most notebooks primarily use `pandas`, `geopandas`, `rasterio` and other standard Python libraries as shown at the top of each notebook.

The model is in R. I suggesting running it in [RStudio](https://www.rstudio.com/products/rstudio/download/).

## Usage

1. Start by merging Yuma deaths with deaths collected by Pima OME.
*Note:* Yuma deaths lat lon were entered into file by hand from pdf.
```bash
../migrant_deaths_analysis/combine_yuma_and_pima_deaths_files.ipynb
```
2. Merge new Trump walls with existing walls (pedestrian fencing).
```bash
../migrant_deaths_analysis/empty_border_by_year/merge_border_wall/merge_new_walls_with_ped_fence.ipynb
```
3. Find sections of empty (no wall or ped fencing) border by corridor by year. This will be important later for the model. *Note:* Make a separate virtual environment to install the `qgis.core` library, using `conda install -c conda-forge qgis`. Follow additional instructions in the
```bash
../migrant_deaths_analysis/empty_border_by_year/empty_border_by_year_by_corridor.ipynb
```
4. Intersect deaths with mountains.
```bash
../migrant_deaths_analysis/mountains/deaths in mountains.ipynb
```
5. Intersect deaths with min and max temp.
```bash
../migrant_deaths_analysis/min_max_temp/min_max_temp_for_each_death.ipynb
```
6. Find distance to nearest road for each death.
```bash
..migrant_deaths_analysis/distance_from_roads_towns_border/roads/Neaest Neighbor using GeoSeries.distance roads .ipynb
```
7. Find distance to nearest town for each death.
```bash
../migrant_deaths_analysis/distance_from_roads_towns_border/towns/Neaest Neighbor using GeoSeries.distance cities towns.ipynb
```
8. Find distance to border
```bash
../migrant_deaths_analysis/distance_from_roads_towns_border/border/Neaest Neighbor using GeoSeries.distance Border.ipynb
```
9. Intersect deaths with cell service. *Note:* Download coverage maps (F477 Mobile Voice Deployment files by provider) found [here](https://www.fcc.gov/general/broadband-deployment-data-fcc-form-477). Reach out to me if you would like the June 2020 files that I used.
```bash
../migrant_deaths_analysis/deaths_in_cell_service/Call service 2020 on all deaths.ipynb
```
10. Combine all data points from previous steps into one file for model and final analysis.
```bash
../migrant_deaths_analysis/deaths_in_cell_service/Call service 2020 on all deaths.ipynb  
```
11. Run final analysis to generate findings
```bash
../migrant_deaths_analysis/final analysis.ipynb
```
12. Run statistical model.
```bash
../migrant_deaths_analysis/compare_negbin_quassi_poisson.Rmd
```
